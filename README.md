# adventofcode-2021

My solutions for [Advent of Code 2021](https://adventofcode.com/2021).

## Getting started

    curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

## Run
Unoptimized:

    cargo run

Optimized:

    cargo run --release

## Run tests
Unoptimized:

    cargo test

Optimized:

    cargo test --release
